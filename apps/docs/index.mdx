# Loop

Loop is an opinionated platform for shipping production-grade AI agents:
deterministic tool calling, governed memory, multi-channel reach, and a
runtime designed for tight latency budgets.

## What is in this site

- **Quickstart** — run the canonical support agent against a real LLM
  in under five minutes.
- **Concepts** — the five primitives every Loop agent is built from:
  agents, tools, memory, channels, and eval.
- **Tutorials** — three end-to-end walkthroughs that take you from a
  blank workspace to a deployed, evaluated agent.
- **Operations** — the security, runbook, and environment-reference
  documentation reviewers and SREs need before going live.
- **API Reference** — the cp-api surface, generated from the OpenAPI
  spec at [`loop_implementation/api/openapi.yaml`](https://github.com/kwameasare/loop/blob/main/loop_implementation/api/openapi.yaml).
- **Cookbook** — copy-pasteable recipes that combine the primitives.

## Where the source of truth lives

This Mintlify site is a curated façade over the canonical engineering
docs in [`loop_implementation/`](https://github.com/kwameasare/loop/tree/main/loop_implementation).
Whenever a page links into the repo, that link is the source of
truth — the pages here mirror just enough to make the public site
useful for partners, customers, and reviewers.
