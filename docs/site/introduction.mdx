# Welcome to Loop

Loop is an open platform for building, evaluating, and operating production AI
agents. Use it to ship a support bot in an afternoon, run rigorous offline
evals, route across LLM providers with a single gateway, and scale to millions
of conversations without rewriting your stack.

## What you get

<CardGroup cols={2}>
  <Card title="Studio" icon="palette" href="/concepts/agents">
    A web app to author, version, and deploy agents — including channels,
    tools, knowledge bases, and eval suites.
  </Card>
  <Card title="Gateway" icon="route" href="/concepts/channels">
    A drop-in OpenAI-compatible gateway with multi-provider routing, retries,
    cost controls, and per-tenant rate limits.
  </Card>
  <Card title="Eval harness" icon="flask" href="/concepts/eval">
    Reproducible offline evals with golden tests, regression budgets, and
    Pareto-front comparisons across providers.
  </Card>
  <Card title="Memory + RAG" icon="database" href="/concepts/agents">
    Short-term, episodic, and long-term memory with vector search and a
    pluggable knowledge-base engine.
  </Card>
</CardGroup>
